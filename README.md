# kvocab 🐹

Маленький тренажер корейської лексики. Працює як статичний сайт — можна відкрити `index.html` локально або захостити на GitHub Pages.

## Режими

- **Переклад слова** — обрати переклад корейського слова з 4 варіантів (після відповіді показуються переклади всіх варіантів).
- **Корейське слово** — обрати корейське слово за перекладом.
- **Введення** — ввести корейське слово з клавіатури; є екранна корейська клавіатура (кнопка ⌨️) зі складанням складів. Якщо ввести інше слово, яке теж є в базі, сайт покаже його значення.
- **Картки** — флеш-картки для перегляду.

Можна обмежити тренування темами (чіпси на головній) та розміром раунду.

## Як додати нові слова

Відкрий `data.js` і допиши об'єкт у масив `VOCAB`:

```js
{"ko": "사과", "tr": "яблуко", "topics": ["Їжа"]},
{"ko": "달리다", "tr": "бігти", "topics": ["Дієслова", "Хобі"], "pol": "달려요", "note": "необов'язковий коментар"},
```

Поля:
- `ko` — корейське слово (обов'язково)
- `tr` — переклад (обов'язково)
- `topics` — список тем (слово може мати кілька); нова тема автоматично з'явиться у списку
- `note` — коментар, показується після відповіді (необов'язково)
- `pol` — розмовно-ввічлива форма (-아/어요) для дієслів і прикметників (необов'язково)
- `alts` — масив альтернативних написань, які приймаються у режимі введення (необов'язково)

## GitHub Pages

Settings → Pages → Deploy from branch → `main`, тека `/ (root)`.

Оригінальні фото слів лежать у `sources/` і не потрапляють у git (`.gitignore`).
